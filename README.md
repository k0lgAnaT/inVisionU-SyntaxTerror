**Инновационная система отбора талантов для университета inVision U с применением Explainable AI (XAI) и NLP.**

---

## 🌟 Наша Миссия (Соответствие ТЗ)

inVision U ищет не просто отличников, а будущих лидеров и агентов изменений. Традиционный отбор страдает от предвзятости, сложности анализа эссе и растущего потока заявок, усиленного генеративным ИИ.

Наше решение — это **Интеллектуальный мост** между кандидатом и приёмной комиссией:
1. **Human-in-the-Loop**: ИИ не принимает решений, он подсвечивает потенциал.
2. **Борьба с Bias**: Режим "Слепого отбора" (Blind Mode) исключает дискриминацию.
3. **Детекция GPT**: Анализ структуры текста для выявления неискренности и ИИ-генерации.
4. **Объяснимость (XAI)**: Каждый балл (Лидерство, Мотивация, Рост) обоснован конкретными сигналами из анкеты.

## 🚀 Функционал системы

### 🏛 Для приёмной комиссии (Админ-панель)
- **Dashboard**: Общая статистика, топ кандидатов и флаги риска.
- **Leaderboard**: Ранжирование по 4 профилям скоринга (Balanced, Leadership, Authenticity, Growth).
- **Candidate Profile**: Глубокий разбор — радар измерений, ИИ-тепловая карта эссе, умные вопросы для собеседования.
- **XAI Validation**: Научное обоснование модели (Spearman ρ, сравнение с простым скорингом).

### 🎓 Для студентов (Портал абитуриента)
- **3-ступенчатая воронка**: Тест -> Документы -> Интервью.
- **Интерактивный календарь**: Выбор слота для Zoom-интервью в реальном времени.

### ⚡ Демо-функции
- **Live Scoring**: Введите эссе и получите мгновенный пре-скор на странице `/submit`.

- **Frontend**: Next.js 14 (App Router), Tailwind CSS, Framer Motion, Recharts.

## 🔒 Этические обязательства и Fairness

Мы строго следуем пункту 6 ТЗ:
- **Никакой дискриминации**: Система не использует демографические, расовые или экономические данные для оценки.
- **Полная прозрачность**: Мы объясняем почему кандидат получил такой балл.
- **Конфиденциальность**: Персональные данные защищены, обработка текста происходит без передачи во внешние сети.
