# Syntax Terror - AI Module for inVision U Selection System

## О модуле
Это ветка проекта **Syntax Terror**, которая отвечает за **ИИ-компонент системы** для поддержки приёмной комиссии inVision U.  

Модуль предназначен для **первичного анализа документов и тестов поступающих**, автоматического расчёта оценки кандидатов и предоставления рекомендаций членам комиссии.

## Цель
- Создать рабочий прототип ИИ для поддержки приёмной комиссии  
- Сократить нагрузку на членов комиссии  
- Подготовить систему для дальнейшего улучшения ИИ и интеграции с остальной системой  

## Текущий прогресс
- Разработан **прототип ИИ-модуля** на Python
- Пока что ИИ работает на основе простого анализа текста, ключевых слов и заранее заданных критериев
