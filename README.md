# inVision U: AI Candidate Scoring Engine

An intelligent, multi-modal scoring engine for university admissions.

## Setup

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

2. Run the engine:
   ```bash
   python main.py
   ```

## Features
- Hybrid scoring (Semantic + Keyword + Linguistic Features).
- Cross-source consistency checking.
- Authenticity and risk detection.
- Human-in-the-loop decision support.
