# Syntax Terror - Web Module for inVision U Selection System

## О модуле
Это ветка проекта **Syntax Terror**, которая отвечает за **веб-интерфейс и бэкенд системы** для поддержки приёмной комиссии inVision U.  

Модуль позволяет:  
- Загружать документы кандидатов  
- Назначать документы и тесты членам комиссии  
- Оценивать кандидатов  
- Просматривать итоговые отчёты  

## Дизайн и архитектура
- Интерфейс разрабатываеться с использованием **HTML, CSS, JavaScript**  
- Дизайн и структура интерфейса созданы в **Figma**
